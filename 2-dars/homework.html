HTML

HyperText Markup Language

HTML which stands for HyperText Markup Language, is the standard markup language for creating web pages. It used to stucture the content of a web page and define the elements and their relationships.HTML consists of a series of elements or tags that are used to enclose and define different parts of the content

CSS

Cascading Style Sheets

CSS which stands for Cascading Style Sheets, is a stylesheet language used for describing the presentation and visual design of web pages written in HTMl and XML. CSS allows web developers to control how web content is displayed, including its layout, fonts, and spacing.
